InternHub is a frontend web application that displays curated internship opportunities and allows users to apply for roles through a simple, interactive interface.
The project focuses on DOM manipulation, UI design, and client-side data handling using vanilla JavaScript.

OVERVIEW:
InternHub simulates a real-world internship portal where users can:
Browse multiple internship roles
View key details like company, stipend, location, and deadline
Apply for an internship using a form
Store application data locally in the browser

FEATURES:
 Dynamic internship cards generated using JavaScript
 Modern UI with hover effects and smooth animations
 Application form with validation
 Uses localStorage to store submitted applications
 Responsive design 

TECH STACK:
 HTML5 - Structure and semantic layout  
 CSS3 - Styling, layout, animations, responsive design  
 JavaScript - DOM manipulation, event handling  
 Web APIs - localStorage for client-side data storage
